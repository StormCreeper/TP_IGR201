# IGR201 - TP
This is my work for the class IGR201. It is a simple drawing program,
with different tools and a simple GUI in QT and C++.
## Implemented
- Drawing free hand, lines, rectangles and ovals
- Unlimited shapes
- Can choose color
- Can choose pen size
- Icons for tools
